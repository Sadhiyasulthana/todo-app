# 📝 Todo App

A simple and interactive Todo Application built using HTML, CSS, and JavaScript. This application allows users to add, view, and delete tasks easily. The tasks are stored using the browser's Local Storage, so they remain saved even after refreshing the page.

## 🚀 Live Demo
https://sadhiyasulthana.github.io/todo-app/

## 📌 Features
- Add new tasks
- Delete tasks
- Tasks remain saved after page refresh
- Simple and clean user interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Browser Local Storage

## 📚 What I Learned
- DOM Manipulation
- Handling user input
- Working with JavaScript arrays
- Using Local Storage to save data in the browser
